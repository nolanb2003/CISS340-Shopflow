# ShopFlow E-Commerce Website
A full working website that uses a database connection through Aiven MySQL and a website connection through GitHub hosted by Netlify

## Prerequisites:
- Node.js installed
- Git installed
- GitHub account
- Netlify account
- Aiven MySQL database (already created)

## Local Development:
1. Clone this repository
2. Install dependencies: `npm install`
3. Create `.env` file with your database credentials
4. Run locally: `npm run dev`
5. Open http://localhost:8888

## Deployment:
1. Push to GitHub
2. Connect your GitHub repository to Netlify
3. Add environment variables to Netlify ( ones in .env file)
4. Deploy website through Netlify

## Features:
- Product Catalog with more than 20 products and descriptions
- Customer lookup
- Business analytical data
- all functionality is from real time database queries
